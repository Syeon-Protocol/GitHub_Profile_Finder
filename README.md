# GitHub Profile Finder
Search any GitHub user and view their profile info.

## Features
- Fetches live data from GitHub API
- Shows avatar, bio, followers, repos, location
- Error handling for invalid usernames
- Keyboard support (Enter key)

## Tech used
HTML, CSS, JavaScript, GitHub REST API
